# DATOS DE LLEGADA PASAJEROS NACIONAL E INTER
## Datos descargados desde datos abiertos del gobierno nacioanl

## Columnas (11)

| Nombre de la columna | Descripción | Nombre del campo API | Tipo de Dato |
|---|---|---|---|
| ANIO | Periodo cronológico anual en el que se registran las operaciones aéreas. | anio | Texto |
| MES | Periodo cronológico mensual en el que se registran las operaciones aéreas. | mes | Texto |
| AEROPUERTO_OPERACION | Indicador de lugar OACI del aeropuerto de registro de la operación. | aeropuerto_operacion | Texto |
| ORIGEN | Punto de partida de la aeronave. | origen | Texto |
| DESTINO | Punto de llegada de la aeronave. | destino | Texto |
| EMPRESA | Designador de la aerolínea (Comercial). | empresa | Texto |
| EXPLOTADOR | Poseedor del Certificado de Operador Aéreo (AOC). | explotador | Texto |
| TIPO_VUELO | Clasificación por propósito del vuelo. | tipo_vuelo | Texto |
| TRAFICO | Clasificación geográfica del servicio. | trafico | Texto |
| TIPO_OPERACION | Naturaleza del movimiento en pista. | tipo_operacion | Texto |
| TotalOperaciones | Conteo volumétrico de movimientos. | totaloperaciones | Número |
